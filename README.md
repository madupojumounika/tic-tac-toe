# 🎮 Tic Tac Toe Game

A simple, interactive, and visually appealing Tic Tac Toe game built using **HTML, CSS, and JavaScript**. This project features a modern UI with smooth effects, making the classic game more engaging.

---

## 🚀 Features

- 🎯 Two-player gameplay (Player X vs Player O)
- 🎨 Modern UI with transparent design
- ⚡ Fast and lightweight
- 📱 Responsive design (works on different screen sizes)
- 🎉 Winning animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 🎯 Game Rules

- The game is played between two players: **X** and **O**
- Players take turns marking a box
- First player to get **3 in a row** (horizontal, vertical, or diagonal) wins
- If all boxes are filled and no winner → **Draw**
